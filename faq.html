FAQ

Q1. Upon loading the ImproveMyCity_Project I get multiple build errors.

This is most probably because the Google APIs are not installed in your machine. Check if the errors are generated from the absense of the com.google.android.maps package. If this is the case follow Q2 and Q3.

Q2. How to download the Google APIs in my system?

Follow the steps below:

  • Click on Android SDK Manager.
  • Select the tick-box next to Android 2.2 (API 8) and press Install Packages.
  • Restart your Eclipse IDE so as the newly installed packages to become visible.
  • See Q.3 on how to load the Google APIs in your build path.

The application has been tested with Android 2.2 (API 8), so you are advised to use this version.

Q3. How to include Google APIs in my build path?

Follow the steps below:

  • Right-click on ImproveMyCity_Project from the Package Explorer and select properties.
  • Select Android from the options on the left column.
  • The Google APIs option should appear within the Project Build Target frame. Select the tick-box next to Google APIs and press OK. All build errors should disappear.
  • If the Google APIs does not appear within the Projecct Build Target frame, then the Google APIs are not installed in your machine. See Q.2 on how to download them from the Eclipse IDE.
